# AI-Driven Human Capital Management System (Frontend)

This is a **Frontend-Only** React Admin Dashboard for an AI-Driven HCM system. It uses `Vite`, `React`, `Material UI`, and `Recharts`. It simulates all backend actions using local state and mock data.

## Features
- **Recruitment**: Job posting, CV screening (simulated), Candidate profiling & Shortlisting.
- **Interview**: Dynamic Question Generation (simulated AI), Live Interview Simulator, Automated Evaluation.
- **Performance**: Employee tracking, Feedback Prediction (AI model simulation), Reports.
- **Retention**: Attrition Risk Analytics, Early Warning Alerts, Intervention Actions, XAI (Explainability).
- **Compliance**: Bias & Fairness monitoring, Audit Logs.

## Tech Stack
- React 18+ (Vite)
- Material UI v5
- React Router DOM
- Recharts
- MUI DataGrid

## Project Structure
- `src/app`: App setup, Routing, Theme, Layouts.
- `src/pages`: Feature pages grouped by module.
- `src/components`: Shared components (Cards, Charts).
- `src/data`: Mock data files for data simulation.

## How to Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open the URL shown in the terminal (usually `http://localhost:5173`).

## Credentials
- Login Page: Any email/password (Mock Auth).
- Just click **Login**.
